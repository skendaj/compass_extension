## Interaction States

### Hover
- Cards: Scale up slightly, add shadow
- Buttons: Darken color, translate up 1px
- Links: Underline, change color

### Active/Focus
- Inputs: Blue border, shadow
- Buttons: Slight scale down
- Cards: Border color change

### Loading
- Disabled state on buttons
- "Searching..." text
- Cursor: not-allowed

### Empty States
- Centered content
- Large icon (48px)
- Helpful message
- Suggested action

- Sparkle icon: Float animation (3s infinite)
- Card hover: 0.2s cubic-bezier
- Button press: 0.1s ease-out
