# Screen Capture Quick Start Guide

## üöÄ Getting Started

### 1. Installation Complete ‚úÖ

The screen capture feature is already installed and integrated. No additional setup needed!

### 2. How to Use

#### Step-by-Step

1. **Open the Extension**
   - Click the Navify extension icon in your browser

2. **Find the "Capture Text" Button**
   - Located in the search view
   - Blue outlined button with a camera icon

3. **Start Capture**
   - Click "Capture Text"
   - An overlay will appear with a crosshair cursor

4. **Select Area**
   - Click and drag to select the text you want to capture
   - A blue dashed box shows your selection

5. **Wait for Processing**
   - OCR will extract the text (shows progress %)
   - Usually takes 2-5 seconds

6. **Use the Text**
   - Extracted text appears in the search box
   - Edit if needed or search immediately

7. **Cancel Anytime**
   - Press **ESC** to cancel the selection

## üîß Troubleshooting

### Common Issues

**Selection overlay doesn't appear?**
- Refresh the page
- Try clicking the button again

**Text not extracted correctly?**
- Try capturing a smaller area
- Ensure better contrast
- Use larger text size

**Processing takes too long?**
- Large selections take more time
- Try smaller areas
- Check browser console for errors

**Button disabled/not clickable?**
- Wait for previous operation to complete
- Refresh the extension
